HP67 AuftragsApp V2

Start:
1. ZIP entpacken.
2. index.html im Browser öffnen.
3. Auf dem iPhone: Teilen > Zum Home-Bildschirm.

Wichtig:
- Die App speichert lokal im Browser.
- Regelmäßig unter Tools > JSON Backup exportieren.
- Die PDF-Listen sind im ZIP enthalten und zusätzlich als App-Module nachgebaut.
- A4-Druck geht über Listen > Drucken oder unten über Druck.

Dateien:
- index.html = komplette App
- manifest.webmanifest / service-worker.js / icon.svg = optional für PWA-Nutzung
- HP67_Organisationslisten_A4.pdf und HP67_Auftragsliste_A4.pdf = Original-PDFs, falls vorhanden
